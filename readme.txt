== Description ==

Dynamic widget to insert classic social buttons and counters on your sidebar.
Title changes based on page type users are visiting (Home, pages and posts).
Insert a comma separated list of social sites in the order you want and start sharing.
Play with all of the configuration options to reach the best visual appeal.
Never forget to share your posts anymore and start increase visitors for your web site.
Internationalization support via widget admin added.

See also: [Official Blog Post](http://www.strx.it/2010/11/simple-sharing-wordpress-widget/).

== Installation ==

Upload the plugin to your blog, Activate it, add to the sidebar from the Widgets admin panel
